[![Hex Docs](https://img.shields.io/badge/hex-docs-ffaff3)](https://hexdocs.pm/gtsc/)

```sh
gleam add gtsc@1
```

## Usage

```sh
$ gleam build
$ bunx tsc
$ bunx tsc --noEmit --allowJs --checkJs --allowImportingTsExtensions --lib es2022,dom build/dev/javascript/gtsc/gtsc.mjs
```
